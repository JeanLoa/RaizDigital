Resúmen:
Raiz Digital es una aplicación de escaneo de plantas avanzado, con incorporaciones de tecnologías de detección por patrones y análisis de formas y estructuras de plantas diversas.

Características principales:
- Cámara escaner.
- Portafolios de plantas.
- Recomendación de plantas similares.
